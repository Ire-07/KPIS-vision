# 📊 KPI Vision

Aplicación de escritorio para generar dashboards de KPIs a partir de archivos CSV de ventas o leads.

## 🚀 Funcionalidades
- Carga de CSV
- KPIs automáticos
- Gráficos interactivos
- Interfaz gráfica
- Aplicación .exe

## 🛠️ Tecnologías
- Python
- PySide6
- Plotly
- Pandas

## ▶️ Uso
```bash
python index.py
